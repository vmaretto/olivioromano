# Olivio Romano - Avatar Virtuale Olio di Roma IGP

Landing page con avatar virtuale D-ID per il Consorzio Olio di Roma IGP.

## Setup

1. Inserire la `data-client-key` di D-ID in `index.html`
2. Testare in locale: `npx serve .` oppure aprire `index.html`
3. Deploy su Vercel

## Stack

- HTML/CSS vanilla (no framework)
- D-ID Agent Widget per avatar
- Hosting: Vercel
